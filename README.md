### rem 适配：

npm i postcss-pxtorem autoprefixer amfe-flexible --save-dev
postcss-pxtorem 是 PostCSS 的插件，用于将像素单元生成 rem 单位
autoprefixer 浏览器前缀处理插件
amfe-flexible 可伸缩布局方案 替代了原先的 lib-flexible 选用了当前众多浏览器兼容的 viewport
